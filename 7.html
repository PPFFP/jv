كيفية عمل تعليق في JavaScript

يمكنك إضافة تعليقات في ملف JavaScript باستخدام // لتعليق سطر و /* */ لتعليق عدة أسطر.

على سبيل المثال:


// هذا تعليق لسطر واحد

/*
هذا تعليق
لعدة أسطر
*/

تعليقات JavaScript تستخدم لتوضيح الشيفرة والتوثيق، ولا تتأثر بتشغيل الكود.
